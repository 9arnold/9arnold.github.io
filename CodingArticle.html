What is a Jolt transform?

Jolt is an open-source JSON to JSON transformation library written in Java.
Jolt's main features are: Provides a set of transformations to deal with the JSON-to-JSON conversion. Transforms the structure of the JSON data, not data manipulation. Uses a JSON file for the specification.
